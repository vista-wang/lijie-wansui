# Universal Rating

网页端通用评价系统。公开评分匿名展示；写操作在后台以实名记入审计。

- 产品规则见 [`AGENT.md`](./AGENT.md)，初始计划见 [`docs/plans/2026-07-15-initial-plan.md`](./docs/plans/2026-07-15-initial-plan.md)

## 开发

```bash
npm install
npm run dev
```

打开 [http://localhost:3000](http://localhost:3000)。

```bash
npm run lint
npm run build
```
